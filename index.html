Summary: Latitude vs X

The purpose of this project was to pull realtime weather data (as of the date listed on the figures) from randomly generated cities across the world and analyze it. The goal was to find trends between the latitude of the cities and the temperature, humidity, cloudiness and wind speed in those areas which were shown via multiple plots of the data. A table of all this data can be seen in the data section of this website.

After this data was acquired, Matplotlib was used to create all of the plots shown under the Visualizations tag. This site provides all the data, visualizations and analysis of these plots and points out trends and correlations observed.
